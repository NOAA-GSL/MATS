Scatter Plots

A scatter plot is the equivalent of two TimeSeries curves graphed against each other, one on the x-axis and one on the y-axis. To make a scatter plot simply select all the parameters of the first TimeSeries curve and click 'Apply to axis' (make sure the correct axis is selected) and you should see the '*AXIS NOT YET APPLIED' text change to the curve text for the curve you just added.

Do this for both axes and then click 'Add Curve' at the bottom. You can add as many curves as you want, just make sure to double-check that you're always applying to the correct axis.
